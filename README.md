La Calculadora de Sumatoria de Riemann es una herramienta web diseñada para visualizar y calcular aproximaciones de integrales definidas usando sumatorias (rectángulos izquierdos, derechos y centrados).
Combina un frontend interactivo con gráficos en tiempo real y un backend de alto rendimiento escrito en C++.

**Características**

Generación gráfica de rectángulos mediante Chart.js

Cálculo preciso de la sumatoria usando un módulo en C++

Interfaz web intuitiva en HTML + CSS + JavaScript

Backend en Node.js que se comunica con el programa C++

Alta velocidad de cálculo incluso con particiones grandes

**Soporte para funciones matemáticas como:**

polinomios

trigonométricas

exponenciales

combinaciones complejas

**Tecnologías Utilizadas
Frontend**

HTML5

CSS3

JavaScript

Chart.js para la visualización gráfica

Backend

Node.js

JavaScript (Express opcional)

C++ para los cálculos numéricos
